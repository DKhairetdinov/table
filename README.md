# Многопользовательский табличный процессор на React + TypeScript

Многопользовательское веб-приложение типа "Табличный процессор"(spreadsheet).
Каждый зарегестрированный пользователь работает только со своими документами - чужие документы недоступны.
Приложение поддерживает создание, редактирование и удаление личных таблиц с богатой функциональностью ячеек.
